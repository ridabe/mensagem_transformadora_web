Create distinctive, production-grade frontend interfaces with high design quality.

Before coding, commit to a BOLD aesthetic direction:
- Choose a clear tone: brutally minimal, maximalist, retro-futuristic, luxury/refined, playful, editorial, brutalist, art deco, etc.
- Ask: What makes this UNFORGETTABLE?

Then implement working code (HTML/CSS/JS, React, Vue, etc.) that is:
- Production-grade and functional
- Visually striking and memorable
- Cohesive with a clear aesthetic point-of-view

Focus on:
- **Typography**: Distinctive fonts. Never Arial, Inter, Roboto ou system fonts genéricos.
- **Color**: CSS variables, cores dominantes com acentos marcantes.
- **Motion**: Animações CSS para micro-interações e page load com staggered reveals.
- **Layout**: Assimetria, sobreposição, fluxo diagonal, espaço negativo generoso.
- **Backgrounds**: Gradient meshes, texturas, padrões geométricos, sombras dramáticas.

NEVER use generic AI aesthetics: purple gradients on white, predictable layouts, cookie-cutter components.
